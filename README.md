# rn-food-delivery
A React Native with Expo app for food delivery with the following features:
* Restaurant list
* Product list by restaurant
* Product detail bottom sheet
* Cart bottom sheet

## Used Technologies
* React Native
* Expo
* React Navigation
* Typescript
* @gorhom/bottom-sheet
* Redux Toolkit
* RTK Query
